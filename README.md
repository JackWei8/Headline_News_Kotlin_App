###app整体架构基于自创的MVVM(欢迎留言探讨)
###做这个app的目的呢让初学者可以快速开发出一个安卓app

#####用到的第三方库
1. glide 一款强大的图片加载库，谷歌推荐。
2. retrofit2。
3. RxJava2
4. rxbinding
5. fragmentation(可构建单activity多fragment app)
6. RxBus
